# Simulation - 仿真驱动引擎

该目录包含负责协调整体仿真过程的驱动组件。

## 文件说明

- **engine.py**: 
  - `SimulationEngine` 类：整个仿真的中枢。
  - **时钟循环**: 驱动仿真的推进 (`step` 方法)。
  - **冲突检测**: 管理路网中的所有车辆，处理生成速度与位置排布。
  - **数据收集**: 汇总仿真过程中的轨迹数据、异常日志及 ETC 检测记录。
- **spawner.py**: 
  - `VehicleSpawner` 类：负责根据仿真计划（如每小时流量）生成车辆的发车时间时间表。

## 仿真流程

1. **初始化**: 加载配置，初始化路网和发车计划。
2. **迭代推进**: 在每个 `dt` 步长内，依次调用各车辆的更新逻辑。
3. **数据封存**: 仿真结束时，整理所有轨迹点并导出为 JSON 结果文件。
