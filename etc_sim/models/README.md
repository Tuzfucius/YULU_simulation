# 模型模块 (Models)

本目录包含仿真系统的扩展分析模型。

## 模块列表

### etc_anomaly_detector.py
**ETC 门架异常检测器** - 基于门架流水数据识别交通异常。

**核心算法**：
1. **TT-Outlier (行程时间离群检测)** - 行程时间超过 μ+2σ 时触发警报
2. **Flow-Drop (流量不平衡分析)** - 上下游门架流量对比检测事故

### etc_noise_simulator.py
**ETC 噪声模拟器** - 策略模式实现的硬件异常模拟（漏读 3%、重复读 2%、延迟上传、时钟偏移）。

### environment.py
**环境影响模型** - 天气系统（晴/雨/雪/雾）和道路坡度物理修正。

**天气效果**：
| 天气 | 速度因子 | 安全间隔因子 | 可视距离因子 |
|------|---------|-------------|-------------|
| 晴天 | 1.0 | 1.0 | 1.0 |
| 雨天 | 0.8 | 1.4 | 0.7 |
| 雪天 | 0.6 | 1.8 | 0.5 |
| 雾天 | 0.7 | 1.6 | 0.3 |

**坡度公式**：`a_adj = -g × sin(θ) × mass_factor`

### phantom_jam.py
**幽灵堵车检测** - 检测无外部障碍的交通波动性拥堵。

### anomaly.py / reaction.py / response.py / coupling.py / queue.py
车辆异常、反应时间、响应强度、多车道耦合、排队形成模型。
