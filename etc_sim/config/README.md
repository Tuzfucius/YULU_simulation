# Config - 仿真参数配置

该目录定义了仿真系统所需的各类物理参数和默认配置。

## 文件说明

- **parameters.py**: 
  - 定义了 `SimulationConfig` 类，作为整个仿真过程的参数载体。
  - 包含了道路、车辆、仿真步进、换道模型及异常模型的各项关键指标。
- **defaults.py**: 
  - 提供了一套标准化的默认参数，作为系统启动或未指定配置时的兜底方案。
- **colors.py**: 
  - 定义了各类车辆（轿车、客车、货车）及状态（异常、受影响）在前端/可视化中的标准配色。

## 关键参数类别

1. **道路参数**: 长度、车道数、车道宽度、ETC 门架位置等。
2. **车辆行为**: 目标速度、最大加速度、安全跟驰距离 (IDM)。
3. **换道策略**: 换道间隙、强制换道距离、礼让系数 (MOBIL)。
4. **异常系统**: 异常发生比例、全局开始时间、影响扩散范围。
