# 项目界面截图 (Project Screenshots)

此文件夹包含 ETC 交通仿真系统的界面截图，用于在项目主 `README.md` 中进行展示。

## 图片说明

| 文件名 | 功能描述 |
| :--- | :--- |
| `visual_playback.png` | 实时仿真可视化界面，展示车辆运动、ETC门架及车流密度。 |
| `simulation_statistics.png` | 仿真统计面板，提供速度、流量、密度等微观数据的图表分析。 |
| `workflow_editor.png` | 基于 React Flow 的可视化规则引擎编辑器，用于设计预警链路。 |
| `scenario_builder.png` | 场景构建器，支持配置车流构成、驾驶风格、天气及路况。 |
| `time_series_workbench.png` | 时序预测智能工作台，展示历史轨迹数据及训练集配置。 |
| `route_editor.png` | 路径与路网编辑器，用于定义仿真车辆的行驶轨迹。 |
| `file_management.png` | 系统资源与运行记录管理器。 |
| `script_editor.png` | 仿真脚本编辑器与控制台。 |
